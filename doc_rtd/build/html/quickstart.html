Troubleshooting and logs¶

Check both stderr/stdout in your current directory and the stuff in ~/logs.

If you used nohup, look at nohup.out.

Most common problem is messed up white space: - extra tabs in summary files - ^M character from switching between OS’s

Next most common problem is typos: - check your file paths - check the summary file attributes and values
